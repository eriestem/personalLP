# Edward Shim - Personal Portfolio

A modern, interactive personal portfolio website built with HTML5, CSS3, and JavaScript.

## Features

- Animated star field background with shooting stars
- Typed.js typing effects on section titles
- Smooth scroll reveal animations
- Responsive design for all devices
- Glass-effect cards with hover animations

## Sections

- **About**
- **Education**
- **Coursework**
- **Experience**
- **Certifications**
- **Projects**
- **Interests**
- **Travel**
- **Contact** 

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- JavaScript (Canvas API, Intersection Observer)
- Typed.js for typing animations
- Google Fonts (Space Grotesk, Inter, Space Mono)
